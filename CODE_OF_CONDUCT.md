# Code of Conduct / 行为准则

This project follows a simple rule: be respectful and constructive. / 本项目遵循一个简单原则：保持尊重，并提供建设性反馈。

## Our Standards / 行为标准

- Be polite and inclusive. / 保持礼貌与包容。
- Assume good intent. / 默认他人出于善意。
- Provide clear, actionable feedback. / 提供清晰且可执行的反馈。

## Unacceptable Behavior / 不可接受的行为

- Harassment or personal attacks. / 骚扰或人身攻击。
- Discriminatory language. / 歧视性语言。
- Trolling or disruptive behavior. / 挑衅、刷屏或破坏性行为。

## Enforcement / 执行方式

Project maintainers may remove or edit contributions that violate this policy. / 项目维护者可以移除或编辑违反本准则的贡献内容。
