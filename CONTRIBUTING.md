# Contributing / 贡献指南

Thanks for your interest in improving this project. / 感谢你愿意改进本项目。

## How to Contribute / 如何贡献

1. Fork the repo and create a feature branch. / Fork 仓库并创建功能分支。
2. Keep changes focused and small. / 保持改动聚焦且规模适中。
3. Ensure the app builds and runs. / 确认应用可以正常构建和运行。
4. Open a pull request with a clear description. / 提交 Pull Request，并清楚描述改动内容。

## Code Style / 代码风格

- Use consistent formatting with the existing C++/Qt style. / 保持与现有 C++/Qt 代码一致的格式。
- Keep UI labels user-friendly. / 界面文案应清晰、易理解。

## Reporting Issues / 问题反馈

Please include / 请提供：

- OS and Qt/OpenCV versions / 操作系统与 Qt/OpenCV 版本
- Steps to reproduce / 复现步骤
- Expected vs actual behavior / 预期行为与实际行为
